生产者与消费者问题

  • PV操作是用来操作信号量的。

    • P：等待——wait——减法作用——阻塞作用

    • V：释放——signal——加法作用——唤醒作用

    • S.value > 0 ：有空闲CPU；

      S.value = 0：CPU刚好用完；

      S.value < 0：进程等待；

      • P操作
      • V操作

  • 解题思路：

  • 例：爸爸往桌子上每次放一个苹果，儿子每次从桌子上拿一个苹果，放苹果和拿苹果不能同时进行，桌子上最多放10个苹果，用PV操作实现同步互斥。

    • 分析：爸爸——生产者——剩余空间(empty) = 10

      ​ 儿子——消费者——已占用空间(full) = 0

      ​ 信号量S = 1，用来实现互斥。

    • 特征：①容器 ≤ 容量

      ​ ②生产 消费

      • 分析：生产者A——B的剩余空间

        ​ 生产者B——A的剩余空间

        ​ 消费者B——自己邮箱还有多少封信

        ​ full-A：A信箱有多少封邮件

        ​ empty-A：A信箱的容量

        ​ full-B：B信箱有多少封邮件

        ​ empty-B：B信箱的容量

        ​ mutex-A：A信箱的信号量

        ​ mutex-B：B信箱的信号量

        ​ full-A = x ， empty-A = M - X

        ​ full-B = y， empty-B = N - y
